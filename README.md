# Number Guessing Game 🎯

A simple Python game where the user guesses a randomly chosen number between 0 and 100. The program provides hints — "Higher" or "Lower" — after each guess, helping the user reach the correct number.

## 📌 How to Play

1. Run the script.
2. The computer selects a random number from 0 to 100.
3. You guess numbers until you find the correct one.
4. It tells you whether to guess higher or lower.
5. The game ends when you guess the correct number, and it shows your total attempts.

## 🧠 Technologies Used

- Python 3

## ▶️ Run It Yourself

```bash
python guessing_game.py
